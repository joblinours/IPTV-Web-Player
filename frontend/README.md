# IPTV Frontend

Frontend React + Vite pour le lecteur IPTV.

## Démarrage

1. Installer les dépendances:

```bash
npm install
```

2. Copier le fichier d'environnement:

```bash
cp .env.example .env
```

3. Lancer en mode développement:

```bash
npm run dev
```

## Build production

```bash
npm run build
```
